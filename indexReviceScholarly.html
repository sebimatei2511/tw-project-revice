Abstract

In acest document se regasesc aspectele tehnice si interactiunea cu utilizatorul avand in vedere proiectul Revice, realizat in coloborare de catre Alexa George-Catalin, Candrea Rares Lucian si Matei Eusebiu-Andrei pentru Tehnologii Web (2021).

Introducere

Scop

Aplicatia Revice ii permite unui utilizator sa vizualizeze diverse dispozitive in functie de anumite caracteristici (pret, model, brand, etc). Aceste date sunt oferite de catre situri ce pot fi precizate de catre utilizator.

Acest document se adreseaza in principal utilizatorului obisnuit, si vizeaza in special principalele functionalitati ale aplicatiei si modul de interactiune cu utlizatorul, pentru a facilita utilizarea aplicatiei.

Viziune

Scopul aplicatiei este acela de a ii arata utilizatorului diverse dispozitive electronice pe care acesta s-ar putea sa si le doreasca conform anumitor date introduse, acesta putand sa compare doua produse similare intr-un mod mai lejer.

Descriere

Perspectiva

Aplicatia este una de tip web ce poate fi folosita si ca extensie de navigator web.

Functiile Produsului

Mai jos sunt functionalitatile pe care aplicatia ReDat le ofera.

  • Crearea unui cont si inregistrarea unui utilizator
  • Modificarea datelor asociate utilizatorului
  • Alegerea siturilor preferate
  • Vizualizarea celor mai populare produse si statisticile acestora

Mediul de operare

Fiind o aplicatie web, ea va rula pe orice browser modern. Astfel aplcatia va putea rula pe toate sistemele de operare ce suporta un browser cum ar fi: Android, IOS, Windows, Linux, etc.

Functionalitati

Autentificarea

Autentificarea utilizatorului se realizeaza apasand pe butonul login din pagina home si presupune completarea campurilor necesare urmand ca apoi utlizatorul sa apese pe butonul "login" si,

Inregistrarea

Aceasta operatie presupune creare unui nou cont de utlizator. Utilizatorul completeaza campurile necesare, apasa pe butonul "sign in" si, in cazul in care informatiile sunt valide un nou cont este creat.

Vizualizarea celor mai recomandate produse

Bazat pe situile alese de utilizator din setari, utlizatorul va putea vedea in pagina home cele mai recomadate produse sub forma unei liste. Fiecare element al listei va contine informatii referitare la produs iar utilizatorul va putea vedea mai multe tipuri de statistici.

Setarile Utilizatorului

Utilizatorul va putea accesa setari asociate contului sau. Astfel el va putea schimba parola, modifica mailul si alte informatii referitoare la contul personal. Tot in pagina de setari va putea modifica siturile preferate.

Interfata si Motivatie

Interfata aplicatitei este simpla si in acelasi timp moderna deoarece contine concepte curente cum ar fi: colturi rotunjite, folosirea fontului Open Sans, etc, elemente menite sa atraga utilizatorul.

Pentru create layout-ului fiecare pagina s-a structurat folosind flex deoarece se poate construi foarte usor interfete dar si alinierea elementelor interne.

Foile de still css au fost modularizate pentru scalabilitate, fiecare fisier css reprezinta un singur obiect/layout, astfel o modificare(de exemplu la un buton) se face intr-un fisier iar modificarile se propaga oriunde este folosit acel fisier.

Aplicatia este responsive astfel ea poate fi disponibila si pe telefoanele mobile. Pentru a face acest lucru, atunci cand layout-ul ajunge la o latime minima, el se restructureaza pastrand astfel latimea paginii cu cea a ecranului.
